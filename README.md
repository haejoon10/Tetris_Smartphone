# 🎮 모바일 & 데스크톱 지원 테트리스

브라우저에서 즐길 수 있는 **HTML/JS 기반 테트리스 게임**입니다.  
데스크톱과 모바일 모두 지원하며, 점수, 레벨, 배경색 변화, 속도 증가, 배경음악(BGM) 등 다양한 기능을 제공합니다.

---

## 🌟 특징

- 다양한 블록 모양
- 블록이 정확히 바닥에 깔림, 좌우 벽 밖에는 설치되지 않음
- 한 줄 제거 시 **10점 추가**
- **30점마다 레벨 1 상승**
- **레벨 상승 시 블록 기본 속도 1 기준, 레벨당 10씩 속도 증가**
- 한 줄 제거 시 **배경색 랜덤 변경** 및 **화면 반짝임 효과**
- 점수 및 레벨 표시
- **배경음악(BGM) 지원**
- 모바일 터치 지원
  - **좌/우 드래그 → 블록 이동**
  - **회전 버튼 → 블록 회전**
  - **⬇️ 빠른 낙하 버튼 → 블록 한 칸 빠르게 내리기**

---

## 🕹️ 게임 조작

### 데스크톱
- **← / →** : 블록 이동  
- **↑** : 블록 회전  
- **↓** : 블록 한 칸 빠르게 내리기  
- **🎵 음악 버튼** : 배경음악 켜기/끄기  

### 모바일
- **좌/우 드래그** : 블록 이동  
- **회전 버튼 클릭** : 블록 회전  
- **⬇️ 빠른 낙하 버튼 클릭** : 블록 한 칸 빠르게 내리기  
- **🎵 음악 버튼 클릭** : 배경음악 켜기/끄기
